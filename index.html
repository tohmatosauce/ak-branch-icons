Arknights
Class Icons

Font icon set and vectors for Arknights branch icons. Made for non-commercial fair-use purposes, such as content creation related to the Arknights game. See the Github for SVG files.

INSTALLING. Just add <link rel="stylesheet" href="//tohmatosauce.github.io/ak-branch-icons/font/style.css"/> to your HTML. Or, self-host the files by cloning the github and hosting the contents in the font directory.

USAGE. Call an icon with an inline element with the ak tag, style (regular or solid), and name of the icon. For example , <span class="ak ak-solid ak-core"></span>.
